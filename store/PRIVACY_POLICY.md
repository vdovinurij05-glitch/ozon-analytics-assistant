# Privacy Policy / Политика конфиденциальности

**Ozon Analytics Assistant**

Дата вступления в силу: 5 февраля 2026 г.

---

## 1. Введение

Настоящая Политика конфиденциальности описывает, как расширение "Ozon Analytics Assistant" (далее — "Расширение") собирает, использует и защищает информацию пользователей.

Разработчик: @first_seller (Telegram)

---

## 2. Какие данные мы собираем

### 2.1 Данные, предоставляемые пользователем:
- Email адрес (при регистрации)
- API ключ сервиса

### 2.2 Данные, собираемые автоматически:
- Данные со страниц Ozon (таблицы, метрики, заголовки) — **только при активном использовании чата**
- URL посещаемых страниц на ozon.ru и seller.ozon.ru
- История сообщений в чате с AI
- Статистика использования (количество запросов, расход токенов)

### 2.3 Данные, которые мы НЕ собираем:
- Пароли от аккаунтов Ozon
- Персональные данные покупателей
- Банковские данные
- Данные с других сайтов

---

## 3. Как мы используем данные

Собранные данные используются исключительно для:
- Предоставления AI-анализа данных Ozon
- Сохранения истории чата для улучшения контекста
- Расчёта и списания баланса за использование
- Улучшения качества сервиса

---

## 4. Хранение данных

- Данные хранятся на защищённых серверах
- История чата хранится до 90 дней
- Пользователь может запросить удаление данных

---

## 5. Передача данных третьим лицам

Мы передаём данные только:
- **Anthropic (Claude AI)** — для обработки запросов искусственным интеллектом

Мы НЕ продаём и НЕ передаём данные:
- Рекламным сетям
- Маркетинговым компаниям
- Другим третьим лицам

---

## 6. Безопасность

Мы применяем следующие меры защиты:
- Шифрование данных при передаче (HTTPS)
- Хэширование API ключей
- Ограничение доступа к серверам

## 7. Права пользователя

Вы имеете право:
- Запросить копию своих данных
- Запросить удаление данных
- Отозвать согласие на обработку

Для реализации прав свяжитесь с нами: @first_seller в Telegram

# Privacy Policy (English)

**Ozon Analytics Assistant**

Effective Date: February 5, 2026

## 1. Introduction

This Privacy Policy describes how the "Ozon Analytics Assistant" extension collects, uses, and protects user information.

Developer: @first_seller (Telegram)

## 2. Data We Collect

### Data provided by user:
- Email address (during registration)
- Service API key

### Automatically collected data:
- Data from Ozon pages (tables, metrics, headers) — only during active chat usage
- URLs of visited pages on ozon.ru and seller.ozon.ru
- Chat history with AI
- Usage statistics (number of requests, token consumption)

### Data we DO NOT collect:
- Ozon account passwords
- Customer personal data
- Banking information
- Data from other websites

## 3. How We Use Data

Collected data is used exclusively for:
- Providing AI analysis of Ozon data
- Saving chat history for context improvement
- Calculating and deducting balance for usage
- Improving service quality

## 4. Data Storage

- Data is stored on secure servers
- Chat history is stored for up to 90 days
- Users can request data deletion

## 5. Third-Party Data Sharing

We share data only with:
- **Anthropic (Claude AI)** — for AI request processing

We DO NOT sell or share data with:
- Advertising networks
- Marketing companies
- Other third parties

## 6. Security

We implement the following security measures:
- Data encryption during transmission (HTTPS)
- API key hashing
- Server access restrictions

## 7. User Rights

You have the right to:
- Request a copy of your data
- Request data deletion
- Withdraw consent for processing

To exercise your rights, contact us: @first_seller on Telegram

## 8. Policy Changes

We may update this policy. The current version is always available at this address.

## 9. Contact

For privacy questions:
- Telegram: [@first_seller](https://t.me/first_seller)
