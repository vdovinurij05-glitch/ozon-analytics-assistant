# Ozon Analytics Assistant

Chrome-расширение для анализа данных на страницах аналитики Ozon с помощью Claude AI.

## Возможности

- Плавающая кнопка на страницах seller.ozon.ru
- Автоматический сбор данных со страницы (таблицы, метрики, графики)
- Чат с AI для анализа данных
- Ответы на вопросы о продажах, выручке, товарах и других показателях

## Установка

### 1. Сгенерируйте иконки

1. Откройте файл `icons/generate-icons.html` в браузере
2. Скачайте все три иконки (16, 48, 128 пикселей)
3. Сохраните их в папку `icons/` с именами: `icon16.png`, `icon48.png`, `icon128.png`

### 2. Установите расширение в Chrome

1. Откройте Chrome и перейдите на `chrome://extensions/`
2. Включите "Режим разработчика" (переключатель справа вверху)
3. Нажмите "Загрузить распакованное расширение"
4. Выберите папку `ozon-analytics-assistant`

### 3. Настройте API ключ

1. Нажмите на иконку расширения в панели Chrome
2. Введите ваш Claude API ключ (получить на [console.anthropic.com](https://console.anthropic.com/))
3. Нажмите "Сохранить"

## Использование

1. Откройте любую страницу аналитики на seller.ozon.ru
2. В правом нижнем углу появится синяя кнопка с вопросительным знаком
3. Нажмите на неё, чтобы открыть чат
4. Задайте вопрос о данных на странице

### Примеры вопросов:

- "Какой товар продаётся лучше всего?"
- "Сравни выручку за эту неделю с прошлой"
- "Какие товары на исходе?"
- "Дай рекомендации по увеличению продаж"
- "Проанализируй эффективность рекламных кампаний"


## Требования

- Google Chrome (или Chromium-браузеры)
- API ключ Anthropic (Claude)

## Стоимость

Расширение использует Claude API. Стоимость зависит от объёма данных и количества запросов.
Примерная стоимость одного запроса: $0.01-0.05

## Безопасность

- API ключ хранится локально в Chrome Storage
- Данные не отправляются на сторонние серверы
- Все запросы идут напрямую в Anthropic API
